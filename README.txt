FOUR YOU Website Template
=========================

Isi folder:
- index.html      : struktur website
- style.css       : tampilan responsive desktop dan mobile
- script.js       : filter produk, modal detail, menu mobile, dan CTA WhatsApp
- assets/         : foto kaos, logo, foto model, dan video brand

Cara pakai:
1. Upload semua isi folder ini ke hosting/public_html atau subfolder website.
2. Buka index.html di browser untuk preview lokal.
3. Edit harga dan nama produk di file script.js.
4. Edit teks website di index.html.
5. Ganti nomor WhatsApp di script.js jika diperlukan. Nomor saat ini: 6285880487484.
6. Untuk deploy di domain, upload folder assets, index.html, style.css, dan script.js secara bersamaan.

Catatan SEO:
- Ganti meta description di index.html sesuai kata kunci utama brand.
- Tambahkan URL website asli pada og:url dan canonical jika sudah punya domain.
- Kompres video jika ukuran website ingin lebih ringan.
